왜 이 책인가

수집→AI 생성 자료는 많지만, 브라우저 자동발행을 24/7 안 죽게 운영하는 법(anti-bot·세션·1회용 임시프로필·멈춤 자가복구·내구성 큐)은 자료가 거의 없습니다. 이 책은 실제로 그 문제를 푼 시스템의 기록입니다.

목차 (6장)

  1. 1. 왜 어려운가 — Tistory API 종료, 자동발행 난관 전체 지도
  2. 2. 콘텐츠 엔진 — Gemini로 검색되는 글 자동생성(검색어 타겟팅·SEO 스니펫)
  3. 3. 발행 엔진 — undetected-chromedriver로 Tistory 에디터 다루기
  4. 4. 안정화 — anti-bot·세션/2FA·1회용 임시프로필·멈춤 자가복구 핵심
  5. 5. 무인 운영 — 스케줄링·내구성 발행 큐·백오프 재시도
  6. 6. 색인·수익화 현실 — 구글/네이버 색인·광고/CPA 배선 + 솔직한 트래픽·수익 현실
