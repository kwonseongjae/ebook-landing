## 이 책이 다루는 것

"AI 블로그 자동화" 자료는 대부분 *글 생성*에서 끝납니다. 이 책은 그 다음 —
**브라우저 자동 발행을 24/7 안 죽게 운영**하기까지의 엔지니어링을 다룹니다.

> 글을 만드는 건 쉬웠습니다. 사람 없이 매일 안 죽게 발행하는 게 진짜 문제였습니다.

## 목차 (6장)

1. **왜 어려운가** — Tistory API 종료(2024) 이후, 자동 발행 난관 전체 지도
2. **콘텐츠 엔진** — LLM으로 검색되는 글 자동생성(검색어 타겟팅·SEO 스니펫)
3. **발행 엔진** — undetected-chromedriver로 에디터(TinyMCE) 다루기
4. **안정화 (핵심)** — 봇 탐지 대응·세션/2FA·1회용 임시 프로필·멈춤 자가복구
5. **무인 운영** — 스케줄링·내구성 발행 큐·백오프 재시도
6. **색인·수익화 현실** — 구글/네이버 색인·광고/CPA 배선 + 솔직한 트래픽·수익 숫자
